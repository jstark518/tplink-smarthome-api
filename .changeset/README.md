# Changesets

This folder has been automatically generated by [changesets](https://github.com/changesets/changesets).

Run `npx changeset` to add a changeset describing your change. On merge to `main`,
the "Version Packages" PR opened by the release workflow applies pending changesets
(bumping versions + updating changelogs); merging that PR publishes the changed
packages to npm.

See the [common questions](https://github.com/changesets/changesets/blob/main/docs/common-questions.md)
for more.
